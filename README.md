## Installation

### CocoaPods

SwiftTurf is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "SwiftTurf"
```

### Carthage

```
github "AirMap/SwiftTurf" ~> 0.3
```
